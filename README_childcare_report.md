足立区・西新井駅 保育園レポート

ファイル:
- adachi_nishiarai_childcare_report.html  — モバイル向けHTMLレポート（編集可）
- report.css — レポート用スタイル

使い方:
1. プロジェクトのルートでローカルサーバを起動します。例:

```powershell
cd C:\Users\Moriyama\tetris-game
python -m http.server 8000
```

2. ブラウザで次のURLを開きます:

```
http://localhost:8000/adachi_nishiarai_childcare_report.html
```

3. データを埋める方法:
- 認可保育園の空き状況や住所は足立区の公式サイト（足立区 保育所入所）や各園の公式ページで確認し、HTMLの該当箇所に手動で反映するか、自動スクレイピングを依頼してください。

PDF作成:
- ブラウザの印刷機能で「PDFに保存」または「印刷」を選択してください。
- 画面上の「PDF出力（印刷）」ボタンを押すと印刷ダイアログが出ます。
