# Versant Practice Materials

This directory contains practice problem sets and example materials for preparing
for the Versant English test. Files are grouped by test part and can be used
as printable worksheets or imported into spreadsheets/quiz tools.

## Contents

- `read_aloud.txt` – sentences for the Read Aloud exercise.
- `repeat_sentence.txt` – sentences for Repeat Sentence practice.
- `sentence_builds.csv` – scrambled words with correct sentence answers.
- `short_answer.csv` – short questions with model responses.
- `minimal_pairs.csv` – pronunciation pairs for listening discrimination.
- `story_retell.txt` – brief story prompts with key points.

Feel free to modify or expand these sets as you continue your study.
